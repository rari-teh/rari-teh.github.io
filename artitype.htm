ARTITYPE

Download | Source

Artitype is an unofficial genotheta font family designed for legibility. It includes all official characters as designed by artifyber, plus a few stand-in characters that make it suitable for a wide array of applications, from coding to longform text.

A breakdown of the special characters supported by this font can be found here.

Artitype D

Artitype D is a standard font meant for general use. Some of its features are:

  • Almost full compatibility with the official GENOTHETAEX font by artifyber
    • The genotheta numerals are encoded in the Private Use Area in order to make them distinct from the more ubiquitous base- arabic numerals, whose codepoint‌s house digit‌s  to  on GENOTHETAEX
  • OpenType ligatures for all  letters that map to latin digraphs — the  original ones (th, sh, gh, ng, wh, ny, ts) and the  later additions (ch, ph, ck, qu, wr, kn, ps)
  • A narrower space character, for more visually uniform text blocks
  • Optionally,　a　fullwidth　space　character　too,　encoded　at　the　ChJK　ideographic　space　Unicode　codepoint　(U+3000)
  • Unofficial stand-ins for common punctuation marks and symbols (mostly taken from latin)
  • Support for arabic numerals as well as genotheta numerals

Artitype Mono

Artitype Mono is a font intended for use on IDEs, terminal emulators and other environment‌s where a monospaced font is preferrable. It is fully compatible with the Windows Command Prompt. While it includes all digraph letters, the font does not support automatic ligatures.

KittyMath

KittyMath is a small Delphi program that converts numbers between the base- arabic figures and the base- genotheta system. It was made before arti published her own online converter, which would make KittyMath slightly redundant if it werent for the fact that our fonts use different encodings for the duotrigesimal digits. You can find KittyMath on my GitHub.
